Matérias do Curso de ADS

Programação Back-end

Nesta disciplina, você aprenderá a desenvolver a lógica por trás das aplicações. Vamos explorar as linguagens de programação mais utilizadas no mercado, como Python e Java, além de conceitos fundamentais de algoritmos e estruturas de dados.

Tópicos abordados:

  • Python (Sintaxe e Estruturas de Controle)
  • Java (Orientação a Objetos)
  • Desenvolvimento de APIs RESTful
  • Manipulação de Dados
  • Introdução a Algoritmos

Programação Front-end

Aprenda a criar interfaces de usuário atraentes e funcionais. Esta matéria abrange as tecnologias essenciais para o desenvolvimento web, incluindo HTML, CSS e JavaScript, além de frameworks modernos que facilitam a criação de aplicações dinâmicas.

Tópicos abordados:

  • HTML5 (Estrutura e Semântica)
  • CSS3 (Estilização e Layout)
  • JavaScript (Interatividade e Dinamismo)
  • Frameworks como React ou Vue.js
  • Ferramentas de Build e Automação

Análise e Projeto de Sistemas

Explore o ciclo de vida do desenvolvimento de software. Nesta disciplina, você aprenderá a analisar requisitos, projetar sistemas e documentar suas soluções. É uma base essencial para qualquer desenvolvedor que deseja criar aplicações eficientes e escaláveis.

Tópicos abordados:

  • Levantamento de Requisitos
  • Modelagem de Casos de Uso
  • Diagramas UML (Unified Modeling Language)
  • Documentação de Sistemas
  • Metodologias Ágeis (Scrum, Kanban)

Banco de Dados

Aprenda a projetar, implementar e gerenciar bancos de dados. Esta disciplina é fundamental para quem deseja construir sistemas robustos e eficientes, garantindo a organização e a segurança das informações.

Tópicos abordados:

  • Modelagem de Dados
  • SQL (Structured Query Language)
  • Bancos de Dados Relacionais
  • Introdução a Bancos de Dados NoSQL
  • Normalização e Otimização

Desenvolvimento Mobile

Desenvolva aplicações que qualquer um pode baixar! Nesta matéria, você utilizará frameworks modernos para criar aplicações que funcionam em sistemas Android e iOS, do protótipo à publicação na loja de apps.

Tópicos abordados:

  • Fundamentos de UI/UX Mobile
  • Criação de Telas e Navegação
  • Acesso a APIs e Dados
  • Testes em Emuladores e Dispositivos
  • Introdução a Frameworks como Flutter ou React Native

Redes de Computadores

Aprenda a configurar e gerenciar redes de computadores. Esta disciplina é essencial para entender como os sistemas se comunicam, garantindo a segurança e a eficiência das conexões.

Tópicos abordados:

  • Fundamentos de Redes
  • Protocolos de Comunicação (TCP/IP)
  • Configuração de Roteadores e Switches
  • Segurança em Redes
  • Monitoramento e Diagnóstico de Redes

Segurança da Informação

Proteja seus sistemas e dados! Nesta matéria, você aprenderá sobre as melhores práticas de segurança, criptografia e como proteger aplicações contra ameaças cibernéticas.

Tópicos abordados:

  • Fundamentos de Segurança da Informação
  • Criptografia e Autenticação
  • Segurança em Aplicações Web
  • Testes de Penetração (Pentesting)
  • Políticas de Segurança e Compliance

Rede de Computadores

Descubra como as redes conectam dispositivos e permitem a comunicação entre eles. Aprenda sobre protocolos, roteamento e segurança de redes.

Tópicos abordados:

  • Fundamentos de Redes
  • Protocolos de Comunicação
  • Configuração de Roteadores e Switches
  • Segurança em Redes
  • Monitoramento e Diagnóstico de Redes
